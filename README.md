# Shashi Shanker Portfolio

A redesigned portfolio website for Shashi Shanker with a modern dark neon glassmorphism theme.

## Included

- `index.html` - main portfolio page
- `assets/css/theme.css` - custom theme styling
- `assets/js/main.js` - interactivity, Isotope filtering and PDF resume generation
- `assets/img/favicon.svg` - SVG favicon

## Features

- Full new visual theme
- Responsive Bootstrap layout
- Isotope portfolio filtering
- Downloadable PDF resume
- Updated stats: 7+ Years Experience, 20+ Projects Delivered, 3 Organizations, 100% Commitment
- SEO and OpenGraph metadata
- Accessible labels, skip link and semantic sections

## How to run

Open `index.html` directly in a browser, or serve the folder with any static web server.
